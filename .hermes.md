# More Firework Mod — Knowledge Graph

This project has a Graphify knowledge graph at `graphify-out/`.

## How to use it

1. For codebase questions, run `graphify query "<question>"` first — queries the graph instead of reading all source files.
2. `graphify path "A" "B"` for relationship tracing.
3. `graphify explain "NodeName"` for focused explanations.
4. Read `graphify-out/GRAPH_REPORT.md` for architecture overview.
5. After modifying code, run `graphify update .` to keep the graph current.
